# Приложение "Rick and Morty"

<img width="4272" height="3974" alt="image" src="https://github.com/user-attachments/assets/d4e885d3-08bc-432e-a0a9-43d5cc22f1f1" />

Это демонстрационное Android‑приложение, написанное на Kotlin с использованием Jetpack Compose.
Цель проекта — показать примеры работы с архитектурой Clean Architecture, кешированием данных и пагинацией.

## Архитектура

Проект разделён на три уровня:

- **data** — слой работы с сетью и базой данных. Здесь используется Retrofit для обращения к публичному API и Room для локального хранилища. Пагинация реализована через библиотеку Paging 3: для списка персонажей применяется `RemoteMediator`, а для поиска — отдельный `PagingSource`.
- **domain** — содержит модели, интерфейсы репозиториев и use case‑ы. Этот уровень не зависит от Android и отвечает за бизнес‑логику.
- **presentation** — слой пользовательского интерфейса на Jetpack Compose. Для внедрения зависимостей используется Hilt, навигация построена на `NavHostController`.

## Функциональность

- отображение списка персонажей с бесконечной прокруткой;
- поиск и фильтр по имени, статусу, виду и гендеру;
- детальный экран персонажа с информацией об эпизодах;
- кеширование данных в базе Room и работа офлайн;
- подтягивание списка жестом Pull‑to‑Refresh.

## Особенности дизайна

В приложении реализованы несколько нестандартных решений:

- **StarryBackground** — анимированный фон с мерцающими звёздами, написанный на Compose без сторонних библиотек;
- **AutoResizeText** — компонент, который автоматически подбирает размер шрифта, чтобы текст помещался в выделенную область карточки персонажа;
- коллапсирующий блок с аватаром на экране деталей: высота изображения и размер аватара плавно меняются при прокрутке.

Эти элементы делают интерфейс более живым и адаптивным, не усложняя при этом архитектуру приложения.
